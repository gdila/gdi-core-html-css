Browser Defaults

All browsers have different defaults for:

  1. Paddings
  2. Margins
  3. Font sizes
  4. Borders

CSS Normalize/Reset

A CSS Reset stylesheet sets everything to be consistent across different browsers

  1. Eric Meyers CSS Reset
  2. <link rel="stylesheet" type="text/css" href="reset.css">

Google Web Fonts

  1. Easiest way to include web fonts
  2. Link to the google stylesheet like all stylesheets
  3. Separate multiple Google fonts with pipe | character
  4. Fonts do add load time, so use sparingly
  5. Fonts will always render differently across browsers
  6. Add font stylesheets before your actual stylesheet (cascade)
<link href='http://fonts.googleapis.com/css?family=Righteous|Cherry+Swash' rel='stylesheet' type='text/css'>
