ELEMENTOS ALINEADOS Y EN BLOQUE,

  • CSS divide HTML en dos tipos: alineados y en bloques.
  • Despues de los elementos en bloque,los buscadores muestran una nueva linea.
  • Elementos en linea: img, a, br, em, strong
  • Elementos en bloque: p, h1, ul, li, es decir, casi todos los otros elementos

ELEMENTO: Div

  • Elemento a nivel de bloque.Cada nuevo div se representa en una nueva linea.
  • Una division, o seccion del contenido en una pagina de html.
  • Se utiliza para agrupar elementos para darles formato con CSS.
  • Aplicar los IDs and Clases de divs para controlar sus estilos con CSS.
<div>
	<p>Contenido<p>
	<p>Contenido<p>
</div>
<div id="encabezamiento">
	<h1>Encabezamiento principal<h1>
</div>
<div class="sub-contenido">
	<p>Algun otro contenido <p>
</div>

Agrupando elementos con div

  • La etiqueta div se utiliza siempre para agrupar elementos juntos en secciones.
  • Por ejemplo, ¿qué pasaría si queremos que los 2 primeros párrafos de una sección sean alineados a la derecha, que sea purpura y resaltado, pero no queremos ningun otro párrafo sea alineado a la derecha?
  • Nosotros debemos agruparlos, de una manera diferente, con el estilo de un elemento div.

Elemento: Span

  • Elemento alineado. Cada nuevo tramo se representa uno a continuacion del otro, solo se envuelve cuando alcanza el borde del elemento que contiene.
  • Se puede utilizar para aplicar estilos a texto en línea a fin de no romper el flujo de contenido.

Span

El span se usa para aplicar un estilo especifico dentro de la linea

.yellow {
	color:yellow;
}
<p>Paragraph with <span class ="yellow">yellow </span> text.</p>

Paragraph with yellow text.

Pseudo clases, mas CSS por enlace

Cambiamos el formato de un enlace pasando este sobre el mismo mediante el uso de pseudo-clases.


En CSS las pseudo-clases son usadas para añadir un efecto especial a agunos selectores.

Syntax: 
selector:pseudo-class
{
	property:value;
}
Example: 
a:link
{
	text-decoration: none;
}

Pseudo-clases, mas CSS por enlace

a:link  {color:#FF0000;} /* unvisited link */
a:visited {color:#00FF00;} /* visited link */
a:hover {color:#FF00FF;} /* mouse over link */
a:active  {color:#0000FF;} /* selected link */

Nota: a:hover debe escribirse despues de a:link y a:visited para ser efectiva!

Nota: a:active debe venir despues de a:hover en la definicion de CSS para ser efectiva
