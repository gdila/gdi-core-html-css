= Su sitio web

Un sitio web es la manera de presentar al mundo un contenido. Usando HTML y CSS se puede lograr que ese contenido luzca bien.

Que es CSS?

CSS = Cascading Style Sheets

CSS es un lenguaje que permite darle estilo a los elementos de su pagina.

CSS trabaja en conjunto con HTML, pero no es HTML.

Que hace CSS?

Cambia el color, posicion, y medida de los elementos del HTML.

Como luce CSS?

Las reglas de CSS

Las reglas de CSS

selector {
	property: value;
}

Un bloque de codigo de CSS es una regla.

Las reglas comienzan con el selector.

Las reglas tienen un conjunto de propiedades y valores.

Un par propiedad-valor es una declaracion.

CSS Sintaxis

Una declaracion esta compuesta de propiedades y valores que se aplican al HTML.

Las declaraciones terminan con un punto y coma.

Los grupos de declaraciones estan rodeados de llaves.

selector {
	property: value;
	property: value;
	property: value;
}

Selector: Elemento

p {
	property: value;
}

Seleccione todos los elementos de un parrafo.

img {
	property: value;
}

Selecciona todos los elementos con imagenes.

Selector: Id

#footer {
	property: value;
}

Selecciona todos los elementos con id = "footer".

<p id="footer">Copyright 2011</p>

Selector: Clase

.warning {
	color: red;
}

Seleccione todos los elementos con class = "warning".

<p class="warning">Run away!</p>

IDs vs. Clases

ID -- Solo se aplica a un elemento en la pagina web.

El "#" es como dices en CSS "ese es un id."

Clase -- Algunos elementos pueden tener la misma clase.

El "." es como dices en CSS "este es el nombre de una clase."

Selector: Descendente

p em {
	color: yellow;
}

Selecciona todos los elementos "em" dentro del parrafo y aplica un estilo CSS

<p>Este elemento es <em>importante</em></p>

Valores de la propiedad

Cada propiedad debe tener uno o mas valores separados por coma. Significa que si el primero no existe, aplica el segundo.

p {
	color: white;
	background-color: red;
	font-family: Arial, sans-serif;
}

Propiedad: Color

La propiedad color cambia el color de los textos.

p {
	color: red;
	color: #ff0000;
	color: rgb(255, 0, 0);
}

Nombre del Color

Valor Hexadecimal

Valor RGB

Los 17 colores estandares son: aqua, black, blue, fuchsia, gray, grey, green, lime, maroon, navy, olive, purple, red, silver, teal, white, and yellow.

Propiedad: color de fondo

La propiedad color de fondo cambia el color del fondo.

p {
	background-color: black;
	background-color: #000000;
	background-color: rgb(0,0,0);
}

Propiedad: Familia tipografica

La propiedad familia tipografica define el tipo de letra que se va a usar.

p {
	font-family: "Times New Roman";
	font-family: serif;
	font-family: "Arial", sans-serif;
}

Conectando CSS con HTML

3 ways

"En linea"

"Incrustado"

"Externo"

Conectando CSS con HTML: En linea

<p style="color:red">Some text.</p>

Se usa el atributo style="..." de un elemento.

Se dificulta el uso en grandes proyectos.

No es recomendado.

Conectando CSS con HTML: Incrustado

<head>
	<style type="text/css">
		p {
			color: blue;
			font-size: 12px;
		}
	</style>
</head>

Dentro del elemento <head>

Se escribe dentro del elemento <style>.

Solo puede ser usado en un archivo a la vez

Conectando CSS con HTML: Vinculos

<head>
	<link rel="stylesheet" type="text/css" href="style.css">
</head>

Comparte los recursos en varias paginas.

Reduce el tama単o de los archivos y puede guardarse en la cache.

Es facil de mantener en grandes proyectos.

Es el preferido por los desarrolladores!

Manos a la obra!

  • Creen un nuevo archivo .css
  • Adicionen el CSS a la pagina hecha la semana pasada.
  • Adicionen estilos para cambio de colores, color de fondo o letras en las diferentes partes del contenido
  • Prueben usar ids and clases para cambiar elementos especificos

Cascada

La "cascada" define las prioridades de cada regla sobre un mismo elemento. Tambien los distintos tipos de inclusiones tienen preferencias.

p {
	color:blue;
	font-family: 'Helvetica';
}
.red{
	color:red;
}
#special{
	font-family: Arial;
}
<p>Paragraph</p>
<p class ="green">Paragraph</p>
<p class ="red">Paragraph</p>
<p class = "red" id ="special">Paragraph</p>
