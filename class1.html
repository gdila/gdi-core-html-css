Anatomía de una Web

  • Elemento
    • Un componente de HTML
    • Un párrafo, una lista, una tabla, una imagen, etc.
  • Etiqueta
    • Denotan el inicio y el final de un elemento
    • Las etiquetas generalmente se abren y se cierran
    <tagname>Stuff in the middle</tagname>
    <p> This is a sample paragraph.</p>

Una Etiqueta por dentro

Tipo de Elementos HTML

  • Elemento contenedor
    • Un elemento que contiene otro(s) elemento(s) dentro
    • <p>Hola Mundo</p>
<p>Hola <b>Mundo</b></p>
  • Elemento independiente
    • Un elemento que no puede contenter ningun otro elemento dentro
    • <br/>
<img/>

Anatomía de un elemento HTML

  • Atributos
    • Brinda información adicional sobre el elemento
    • Class, ID, language, style, identity, source
    • Se escribe dentro de la etiqueta que abre el elemento
  • Valor
    • Valor es el contenido asignado al atributo
    • El valor debe ser escrito dentro de comillas
    • <div id="copyright">© GDI 2013</div>
<img src="my_picture.jpg" />
<a href="http://girldevelopit.com">GDI</a>

Doctype

El primer elemento de una Web es el Doctype, el cual le dice al Navegador que versión de HTML debe usar.

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
<!DOCTYPE html>

Etiqueta HTML

Después del <!doctype>, la sigiuente etiqueta a escribir siempre debe ser <html>

<!DOCTYPE html>
<html>
</html>

Etiquetas Head y Body

HEAD

Dentro de la etiqueta <heat> se encuenta el título de la Web e información sobre su creador, idioma, etc. El contenido de esta seccion no es visible al visitante.

HEAD

La etiqueta <body> contiene todos los elementos que constituyen la Web en si. Todo lo contenido en el body es visible al visitante.

Etiquetas Head y Body

<!DOCTYPE html>
<html>
	 <head>
		<title>Title of the page </title>
	 </head>
	 <body>
		The page content here.
	 </body>
</html>

Elemento: Vínculo

Los vínculos tienen tres componentes:

  • Etiqueta: <a></a>
  • Dirección: "http://www.girldevelopit.com"
  • Ancla "Girl Develop It"
<a href="http://www.girldevelopit.com">Girl Develop It</a>

Girl Develop It Homepage

Las etiquetas <a> rodean el "Ancla", que es el texto que se muestra en el link

Atributos del link

Los links pueden ser configurados para abrir en otra pestaña, o para enviar un email.

<a href="home.html" target="_blank">Texto ancla</a>

El contenido de este vínculo se abrirá en otra pestaña

<a href="mailto:info@girldevelopit.com">Manda un e-mail!</a>

Este vínculo enviará un email en vez de abrir una página web.

Camino absoluto y relativo del vínculo

  • Relativo
    • Los caminos relativos dependen de la página donde el vínculo se encuentra.
      • Recursos en el mismo directorio de la página. "archivo.jpg"
      • Recursos en subdirectorios después de la página. "imagenes/archivo.jpg"
  • Absoluto
    • Los caminos absolutos se refieren a la localización especifica del archivo, incluyendo el dominio. "https://www.girldevelopit.com/chapters/miami-fort-lauderdale"
      • Mayormente usado cuando se accede a recursos fuera de nuestro dominio.
