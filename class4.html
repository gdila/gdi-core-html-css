Posicionamiento estatico

Los elementos en HTML son posesionados estaticamente de manera predeterminada.

Los elementos estáticos se posicionan en el flujo normal de la página

Los elementos estáticos ignoran las especificaciones de propiedad arriba, abajo, derecha o izquierda.

Posicionamiento Estatico

En un flujo normal, las cajas de linea fluyen de izquierda a derecha, involucrando a la línea siguiente cuando es necesario.

<img src="images/cookie1.png"/>

Posicionamiento Relativo

  • Toma el elemento que esta fuera del flujo normal, y permite que este sea movido a la parte superior, a la izquierda, a la derecha o a la inferior.
  • No afecta a los elementos que estan a su alrededor.
  • Introduce un elemento en un "contexto de posicionamiento" en el cual la posición de otros elementos es relativo a el.
  • Los posicionamientos absolutos y relativos se utilizan juntos.

Posicionamiento Relativo

El valor "relativo" pondra el elemento en el flujo normal, pero lo compensa de acuerdo con las propiedades superior/izquierda/derecha/inferior.

.relative{
	position: relative;
	left: 80px;
	top: 20px;
	height: 100px;
	background-color: yellow;
}

Posicionamiento Absoluto

  • Los elementos posicionados estan afuera del flujo normal.
  • Un elemento absolutamente posicionado es desplazado del bloque que lo contiene, posicionadolo relativamente.
  • Este bloque es el primer elemento que tiene una posicion que no es estatica.
  • Si el elemento no es encontrado,el bloque es <html>.
  • Los otros elementos actuan como si este elemento, no estuviera ahi
  • Determinados por valores compensados en las propiedades arriba, abajo, izquierda, derecha.

Posicionamiento Absoluto

El valor "absoluto" tomara el elemento que esta fuera del flujo normal y lo posicionara en relacion con la ventana (o con el elemento no estatico mas cercano).

.top{
	position: absolute;
	top: -40px;
	right: 10px;
	background-color: yellow
}
.bottom{
	position: absolute;
	bottom: -40px;
	left:60px;
	background-color: green
}

Z-index

Cuando se utiliza el posicionamiento para mover los elementos fuera del flujo normal de contenido, los elementos pueden superponerse. Ud. puede cambiar el orden de superposición con z-index.

El elemento con el mayor z-index va en la parte superior.

.bottom{
	position: absolute;
	bottom: 10px;
	left:60px;
	background-color: yellow;
}
.top {
	position: absolute;
	bottom: 15px;
	left:60px;
	background-color: green;
	z-index: 2;
}

Float

  • "Floating" un elemento es tomarlo de su flujo normal y colocarlo lo mas a la izquierda o a la derecha, que sea posible, del elemento que lo contiene.
  • Cualquier otro elemento, como los parrafos o listas, seran envueltas alrededor del elemento flotante.
  • Se debe especificar siempre la anchura al flotar un elemento, de lo contrario es probable que ocupe toda la página y no aparezca como flotado.
  • Ud. debe especificar a) Si un elemento es float o no, y b) sobre que parte el ﬂoats.

Usamos floats para colocar los elementos lado a lado

Si ud. quiere que dos elementos a nivel de bloque esten uno al lado del otro, ud necesita float ambos elementos. Uno a la izquierda y otro a la derecha.

Clear

  • Clearing nos dice del lugar que toman los elementos(derecha, izquierda o ambas) otros elementos no pueden aparecer.
  • Si ud. tenia una imagen flotando en la izquierda y ud. no quiere que el parrafo aparezca al lado de este, puede agregar clear: left; en el parrafo.
  • Clearing ambos lados asegura que los ﬂoats no pasen por encima de los elementos clear.
clear: right;
clear: left;
clear: both;
